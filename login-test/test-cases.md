# Test Cases - Login

## TC01 - Valid Login
Steps:
1. Enter valid username
2. Enter valid password
3. Click login

Expected:
User logs in successfully 

---

## TC02 - Invalid Password
Steps:
1. Enter valid username
2. Enter wrong password
3. Click login

Expected:
Error message appears

---

## TC03 - Empty Fields
Steps:
1. Leave fields empty
2. Click login

Expected:
Validation message appears
