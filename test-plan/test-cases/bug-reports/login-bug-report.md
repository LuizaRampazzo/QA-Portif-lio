# Bug Report – Login Functionality

## Bug ID
BUG-LOGIN-001

## Title
Login allows submission with empty password field

## Environment
Browser: Google Chrome  
OS: Windows 11  
URL: https://example-login-page.com

## Steps to Reproduce
1. Open the login page
2. Enter a valid email
3. Leave the password field empty
4. Click "Login"

## Expected Result
System should display validation message indicating that the password field is required.

## Actual Result
System attempts to submit the form without password validation.

## Severity
High

## Priority
High

## Status
Open
