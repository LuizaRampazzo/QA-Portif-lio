# Test Plan – Login Functionality

## Objective
Verify that the login functionality works correctly for valid and invalid credentials.

## Scope
Testing the login feature of a web application login page.

## Test Types

- Functional testing
- Negative testing
- Usability testing

## Entry Criteria
User must be able to access the login page.

## Exit Criteria
All test cases executed and documented.

## Risks

- Incorrect authentication handling
- Error messages not displayed properly
- Session management failures
