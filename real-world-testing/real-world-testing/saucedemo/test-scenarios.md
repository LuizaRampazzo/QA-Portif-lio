# Test Scenarios – SauceDemo Login

## Feature
Login Authentication

## Application
https://www.saucedemo.com

---

### SC01 – Login with valid credentials
Verify that a user can successfully log in using valid username and password.

---

### SC02 – Login with invalid password
Verify that the system displays an error message when an incorrect password is entered.

---

### SC03 – Login with invalid username
Verify that the system prevents login when the username is incorrect.

---

### SC04 – Login with empty username
Verify that the system displays a validation message when the username field is empty.

---

### SC05 – Login with empty password
Verify that the system displays a validation message when the password field is empty.

---

### SC06 – Login with both fields empty
Verify that the system prevents login and displays validation messages when both fields are empty.
